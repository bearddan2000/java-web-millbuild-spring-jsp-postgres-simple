# java-web-millbuild-spring-jsp-postgres-simple

## Description
A jsp springboot java gradle build,
that connects to postgres database.

## Tech stack
- springboot
  - web
- jsp
- millbuild
- bootstrap
- jquery
- dataTable

## Docker stack
- postgres
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
